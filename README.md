# WeChat MP Local Exporter

本项目用于在你已登录的微信公众号后台页面中，本地导出「近期发表」文章数据，并可选抓取公开文章页正文，方便做内容复盘、选题分析和账号增长诊断。

它提供两种使用方式：

- `wechat_mp_recent_export.user.js`：Tampermonkey / 油猴脚本。
- `wechat_mp_exporter_extension/`：Chrome 已解压扩展版本。

## 功能

- 导出已发表文章列表。
- 导出阅读数、点赞数、分享数、收藏/评论等页面或接口可见指标。
- 导出文章链接、封面链接、发布时间、标题。
- 可选抓取公开文章页正文文本。
- 支持 CSV 和 JSON。
- 面板可拖动、可折叠。
- 默认单线程、低频间隔、失败退避。

## 隐私与安全边界

项目代码不包含个人敏感信息，也不内置任何账号数据。

脚本不会：

- 保存或导出 Cookie。
- 保存或导出 token。
- 上传数据到第三方服务器。
- 读取浏览器密码、历史记录或本地文件。
- 绕过登录、验证码或权限校验。

脚本会：

- 在你已登录的 `https://mp.weixin.qq.com/` 页面内运行。
- 使用当前页面的登录态发起同源请求，读取公众号后台本来返回给页面的数据。
- 将采集结果保存在浏览器 `localStorage` 中，直到你点击 `Clear`。
- 导出 CSV/JSON 到本机下载目录。

更详细说明见 [安全说明](docs/security.md)。

## 快速开始：油猴脚本

1. 安装 Tampermonkey。
2. 新建脚本。
3. 粘贴 `wechat_mp_recent_export.user.js` 的内容并保存。
4. 打开微信公众号后台「近期发表」页面。
5. 右下角出现 `MP Exporter` 面板。
6. 优先点击 `API all CSV` 导出完整文章列表。
7. 如果需要正文，点击 `API+Content`。

## 快速开始：Chrome 扩展

1. 打开 `chrome://extensions`。
2. 开启「开发者模式」。
3. 点击「加载已解压的扩展程序」。
4. 选择 `wechat_mp_exporter_extension/` 目录。
5. 打开微信公众号后台「近期发表」页面后使用面板。

扩展安装说明见 [使用手册](docs/manual.md)。

## 面板按钮

- `API all CSV`：通过后台列表接口按分页导出全部文章数据。
- `API+Content`：先导出全部文章数据，再逐篇抓取正文文本，最后导出 CSV。
- `Pages CSV`：模拟点击分页按钮并扫描页面 DOM，适合作为 API 模式失败时的备用方案。
- `Content CSV`：基于已采集到的文章链接抓正文并导出 CSV。
- `Content JSON`：抓正文并导出 JSON，JSON 中会包含正文 HTML。
- `Scan`：扫描当前可见页面。
- `Scroll`：滚动扫描当前页面。
- `CSV` / `JSON`：导出当前已采集数据。
- `Stop`：停止当前采集任务。
- `Clear`：清空浏览器本地已采集缓存。

## 默认采集节奏

为了避免短时间内产生大量请求，脚本默认使用温和节奏：

- 列表分页：每页等待 `5-12` 秒。
- 文章正文：每篇等待 `9-22` 秒。
- 翻页点击：每页等待 `4-9` 秒。
- 滚动扫描：每轮等待 `2.5-6.5` 秒。
- 请求异常：等待 `45-90` 秒后重试，最多重试 2 次。

这些参数在脚本顶部可以自行调整。

本项目面向个人或团队导出自己有权限访问的微信公众号后台数据，用于本地分析和备份。请遵守微信公众平台规则，不要用于未授权的数据采集或绕过平台限制。
